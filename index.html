A small library
of quiet loaders.

Twenty‑eight dot‑matrix patterns, each animated from a single CSS keyframe and a per‑dot delay map. No noise sampling, no GIFs, no JavaScript runtime. Drop the SVG into a page and it loops forever — about four kilobytes a piece.

Notes

Each loader is a self‑contained <svg> with an inline <style> block. Copy one out and paste it anywhere — there are no external dependencies and no required font. They respect prefers-reduced-motion and fall back to a calm static state.
